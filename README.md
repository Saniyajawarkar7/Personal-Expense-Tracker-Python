# Personal Expense Tracker (Python GUI)

A GUI-based personal expense tracker built using Python that helps users record, manage, and analyze daily expenses with interactive charts and statistical insights.

## Features

- Add and categorize daily expenses
- View and delete individual expense records
- Visualize spending by category (Pie Chart)
- Monthly spending trend analysis (Line Chart)
- Expense statistics using NumPy (Average, Highest, Lowest, Total)
- CSV-based data storage
- Clean GUI built with CustomTkinter

## Technologies Used

- Python
- CustomTkinter
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure

expense_tracker.py → Main application  
expenses.csv → Data storage file  
requirements.txt → Required libraries
